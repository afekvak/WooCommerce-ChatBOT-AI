WooCommerce MCP — Full CRUD + Analytics + AI Tool Ideas
===============================================================

what to do when adding a tool:
1. first of all create a function that fetches information using api in WooCRUD.ts
2. Add a formatting function in utils/formatWoo.ts
3. add schemas/args and register mcp tool in tools/woo folder depends on the crud types
4. add quick/intent rules
5. check if the AI calls the tool correctly using the widget
6. when the tool works correctly checkmark it in the list below


SECTION 1: PRODUCTS – FULL CRUD & QUERY VARIANTS
------------------------------------------------
✅✅ createProduct — Create a new product
✅✅ updateProductById — Update product by numeric ID
✅✅ updateProductBySKU — Update product by SKU
- deleteProductById — Delete product by numeric ID
- trashProductById — Move product to trash
- restoreTrashedProductById — Restore product from trash
✅✅ getProductById — Fetch single product by ID
✅✅ getProductBySku — Fetch product by SKU
✅✅ getProductsByCategory — List products inside a category
✅✅ getProductsByName — Fetch products array with the name
- getProductsByTag — List products filtered by tag
- getProductsByAttributes — Filter by attributes (color, size, etc.)
- getProductsByStatus — Query by draft, pending, private, published
- getProductsByStockStatus — Query by instock / outofstock / backorder
- getProductsBySearchQuery — Search products by name or description
✅✅ getAllProducts — Fetch all products (with pagination) 
- getVariableProductChildren — Fetch variations of a specific product
- createProductCategory — Create a new product category
- updateProductCategory — Edit category details (name, slug, parent)
- deleteProductCategory — Delete category
- getAllProductCategories — Retrieve all categories
- uploadProductImage — Upload image to a product
- removeProductImage — Delete an image from a product gallery
- setFeaturedImage — Set the featured image for a product
- duplicateProduct — Clone an existing product
✅✅ bulkUpdatePrice
